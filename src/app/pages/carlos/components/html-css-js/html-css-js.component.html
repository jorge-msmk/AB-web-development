Apuntes sobre HTML, CSS y JavaScript

HTML (HyperText Markup Language)

HTML es el lenguaje de marcado estándar para crear páginas web. Se utiliza para estructurar el contenido en la web y definir elementos como encabezados, párrafos, enlaces, imágenes y más.

Características principales:

  • Estructura el contenido web.
  • Utiliza etiquetas (<tag>) para definir elementos.
  • Es el componente básico de cualquier página web.
  • Permite incrustar otros lenguajes como CSS y JavaScript.

Etiquetas comunes:

  • <!DOCTYPE html>: Define el tipo de documento.
  • <html>: Contiene todo el contenido HTML.
  • <head>: Contiene metadatos y enlaces a CSS/JS.
  • <title>: Define el título de la página.
  • <body>: Contiene el contenido visible de la página.
  • <h1> - <h6>: Encabezados de diferentes niveles.
  • <p>: Define un párrafo.
  • <a>: Define un enlace.
  • <img>: Inserta una imagen.

CSS (Cascading Style Sheets)

CSS es un lenguaje de estilos utilizado para describir la presentación de un documento HTML. CSS controla el aspecto visual de los elementos HTML, incluyendo colores, fuentes, márgenes, disposición y más.

Características principales:

  • Separa la presentación del contenido.
  • Permite aplicar estilos a múltiples páginas web de manera eficiente.
  • Utiliza selectores y reglas para definir estilos.
  • Soporta diseños responsivos mediante media queries.

Conceptos clave:

  • Selectores: Identifican los elementos HTML a los que se aplicarán los estilos. Ejemplos: elemento, #id, .clase.
  • Propiedades: Definen los estilos que se aplicarán a los elementos seleccionados. Ejemplos: color, font-size, margin.
  • Valores: Especifican el valor de una propiedad. Ejemplos: red, 16px, 10px.
  • Box Model: Modelo de caja que describe el diseño y el espaciado de los elementos HTML.

JavaScript (JS)

JavaScript es un lenguaje de programación que se utiliza para crear contenido web interactivo. JS permite agregar dinámicas a las páginas web, como responder a eventos del usuario, validar formularios, crear animaciones y mucho más.

Características principales:

  • Lenguaje de programación orientado a objetos.
  • Se ejecuta en el navegador del cliente.
  • Permite la manipulación del DOM (Document Object Model).
  • Soporta programación asíncrona mediante promesas, async/await.

Conceptos clave:

  • DOM (Document Object Model): Representación estructurada del documento HTML que permite manipular su contenido y estructura.
  • Eventos: Mecanismo que permite ejecutar código JS en respuesta a acciones del usuario (como clics, teclas presionadas, etc.).
  • Funciones: Bloques de código reutilizables que realizan una tarea específica.
  • AJAX: Técnica para realizar solicitudes asíncronas al servidor sin recargar la página.
  • API (Application Programming Interface): Conjunto de funciones y procedimientos que permiten la creación de aplicaciones que acceden a las características o datos de un sistema operativo, biblioteca o servicio.
