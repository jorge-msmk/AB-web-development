Apuntes Completos sobre HTML, CSS & JavaScript

Fundamentos de HTML

  • Estructura básica:

    Cada documento HTML comienza con una declaración de tipo de documento "!DOCTYPE html". La estructura incluye elementos "html, head y body".

  • Elementos esenciales:

    Etiquetas como "h1-h6" para encabezados, "p" para párrafos, "a" para enlaces, y "img" para imágenes son cruciales para la creación de contenido.

  • Listas:

    HTML ofrece listas ordenadas "ol" y desordenadas "ul", con elementos de lista "li" para organizar contenido.

  • Tablas:

    Se construyen con table, definiendo filas con "tr", celdas con "td" y encabezados de celda con "th".

  • Formularios:

    Para la interacción del usuario con form, utilizando "input", "label", "textarea", y "button" para capturar datos.

Aspectos Avanzados de CSS

  • Herencia y Cascada:

    Los estilos definidos en CSS pueden heredarse de un elemento padre a un hijo, y las reglas se aplican según la especificidad y el orden de declaración.

  • Animaciones y Transiciones:

    Utilizando las propiedades animation y transition para añadir efectos dinámicos a los elementos de la página.

  • Media Queries:

    Permiten adaptar la presentación del contenido a diferentes tipos de dispositivos y tamaños de pantalla, usando {{"@media"}}.

  • Grid Layout:

    Un sistema de diseño bidimensional que permite manejar filas y columnas, proporcionando un control más fino sobre los layouts complejos.

  • Variables CSS:

    Las variables (propiedades personalizadas) permiten reutilizar valores en diferentes partes de las hojas de estilo, declaradas como --mi-variable.

Conceptos Clave en JavaScript

  • Sintaxis y Tipos:

    JavaScript es un lenguaje dinámico con tipos como números, cadenas, booleanos, objetos, y más. Las declaraciones pueden hacerse con var, let, o const.

  • Funciones:

    Unidades de código reutilizable que se pueden definir como funciones regulares, expresiones de función, o como funciones flecha (arrow functions).

  • Manipulación del DOM:

    Permite interactuar y modificar elementos de la página utilizando métodos como getElementById(), querySelector(), y otros.

  • Asincronía:

    JS maneja operaciones asincrónicas usando callbacks, promesas, y async/await para manejar comportamientos no bloqueantes.

  • Frameworks y Librerías:

    Herramientas como React, Angular, y Vue ayudan a construir aplicaciones robustas y escalables usando patrones modernos de desarrollo.
