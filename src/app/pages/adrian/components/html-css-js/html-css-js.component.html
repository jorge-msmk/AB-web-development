Apuntes Básicos

HTML

HTML (HyperText Markup Language) es el estándar para crear páginas web. Es un lenguaje de marcado que define la estructura de una página web mediante el uso de elementos y etiquetas.

<!DOCTYPE html>
<html>
<head>
    <title>Título de la página</title>
</head>
<body>
    <h1>Encabezado</h1>
    <p>Este es un párrafo.</p>
</body>
</html>

CSS

CSS (Cascading Style Sheets) es un lenguaje de estilos utilizado para describir la presentación de documentos HTML. CSS controla el diseño, el color, las fuentes y el espaciado de los elementos en una página web.

body 
    font-family: Arial, sans-serif;
    line-height: 1.6;


h1 
    color: #333;


p 
    color: #666;
