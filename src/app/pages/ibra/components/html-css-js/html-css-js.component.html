Apuntes de HTML, CSS y JavaScript

HTML (HyperText Markup Language)

HTML es el lenguaje estándar para crear páginas web. Utiliza una estructura de etiquetas para definir el contenido de la página.

Conceptos básicos:

  • Etiqueta: Representa un elemento HTML, escrita entre <>. Ejemplo: <p>
  • Atributos: Proporcionan información adicional sobre un elemento, escritos dentro de la etiqueta. Ejemplo: <a href="https://example.com">
  • Estructura básica: 
    <!DOCTYPE html>
<html>
<head>
    <title>Título de la página</title>
</head>
<body>
    <h1>Encabezado</h1>
    <p>Párrafo de texto</p>
</body>
</html>

CSS (Cascading Style Sheets)

CSS es el lenguaje utilizado para describir la presentación de documentos HTML. Permite aplicar estilos como colores, fuentes y espaciamiento.

Conceptos básicos:

  • Selección de elementos: Utiliza selectores para aplicar estilos a elementos específicos. Ejemplo: p color: red;
  • Propiedades y valores: Cada estilo se define con una propiedad y su valor. Ejemplo: color: blue;
  • Incorporación de CSS:
    • En línea: <p style="color: red;">Texto</p>
    • Interno: Dentro de una etiqueta <style> en el <head>. Ejemplo: 
      <style>
p 
    color: green;

</style>
    • Externo: En un archivo CSS separado enlazado con <link>. Ejemplo: <link rel="stylesheet" href="styles.css">

JavaScript

JavaScript es un lenguaje de programación que permite crear contenido dinámico y interactivo en las páginas web.

Conceptos básicos:

  • Incorporación de JavaScript:
    • Interno: Dentro de una etiqueta <script>. Ejemplo: 
      <script>
console.log("Hola, mundo!");
</script>
    • Externo: En un archivo JS separado enlazado con <script>. Ejemplo: <script src="script.js"></script>
  • Variables: Utilizadas para almacenar datos. Ejemplo: var nombre = "Juan";
  • Funciones: Bloques de código que realizan una tarea específica. Ejemplo: 
    function saludar() 
    alert("Hola, mundo!");

saludar();
  • Eventos: Acciones que pueden ser detectadas por JavaScript, como clics. Ejemplo: 
    <button onclick="saludar()">Haz clic</button>
