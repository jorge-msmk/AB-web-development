Apuntes de HTML, CSS y JavaScript

HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Se utiliza para estructurar y dar formato al contenido de las páginas web.

  • Elementos básicos:
    • <html> - Define el documento HTML
    • <head> - Contiene metadatos/información sobre el documento
    • <title> - Establece el título del documento
    • <body> - Define el cuerpo del documento, contiene el contenido visible
  • Etiquetas comunes:
    • <p> - Párrafo
    • <a> - Enlace
    • <img> - Imagen
    • <ul>, <li> - Lista desordenada y elementos de lista

CSS

CSS (Cascading Style Sheets) se usa para describir la presentación de un documento HTML. Permite separar el contenido de la presentación visual.

  • Selector básico:
    • elemento { propiedad: valor; } - Estilo para un elemento
  • Propiedades comunes:
    • color - Color del texto
    • background-color - Color de fondo
    • font-size - Tamaño de la fuente
    • margin - Espacio alrededor de los elementos

JavaScript

JavaScript es un lenguaje de programación que permite crear contenido dinámico en las páginas web. Se utiliza para hacer páginas interactivas.

  • Declaración de variables:
    • var - Declara una variable (obsoleta en ES6+)
    • let - Declara una variable con ámbito de bloque
    • const - Declara una constante
  • Funciones:
    • function nombreFuncion() { ... } - Declara una función
  • Manipulación del DOM:
    • document.getElementById('id') - Selecciona un elemento por su ID
    • document.querySelector('selector') - Selecciona el primer elemento que coincide con el selector
